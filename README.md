# Spring Boot Board (SBB) - JDBC Template Refactoring

> **'점프 투 스프링부트'** 실습 프로젝트를 기반으로 하며, **JPA(ORM)를 사용하지 않고 JDBC Template을 활용하여 직접 SQL을 매핑함**

<br>

##  Tech Stack
- **Language**: Java 17
- **Framework**: Spring Boot 4.0.1
- **Database**: H2 
- **Data Access**: **Spring JDBC Template** (No JPA)
- **Frontend**: Thymeleaf, Bootstrap 5, Custom CSS
- **Build Tool**: Gradle

<br>

## Key Features (구현 기능)
- 질문, 답변 등록 가능
- 회원가입, 로그인 구현
